Allows setting a page state "Broken" or "Works" for bp-wiki pages

== Description ==
This plugin allows setting a page state "Broken" or "Works" for <a href='http://buddypress.org/community/groups/bp-wiki'>bp-wiki</a> pages.
<h3>Features:</h3>
<ul>
<li>Adds widget to set the state on wiki pages</li>
<li>Widget displays wiki stats and broken pages list in wiki root</li>
</ul>

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `bp-wiki-pagestate` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
